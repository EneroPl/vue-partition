# About package

# What is it?

Partitioning is the division of stored database objects into separate parts with separate physical storage parameters. This component represents the partitioning visualization. Vue component representing:

  • Visual representation of data in the form of sections (as disk space) with customization of each object;
  • Flexible settings for data editing with flexible responsive customization of behavior properties.

# Where is it useful?

You can use the component in a situation where you need to represent data in the form of partitions, as in "windows" storage spaces. Also, you can easily set up a component to edit individual data.
