# Getting started

# Installation

Now you can install package from npm only:

npm i @eneropl/vue-partition

Yarn package

Currently, this package is not provided by Yarn. This possibility will be implemented in the near future. If you need to use the package without using npm, then you can refer to the package source (opens new window)

# Usage

Once the package is installed, you can import it into your component:

<template>
   <VuePartition />
</template>

<script>
 import { VuePartition } from '@eneropl/vue-partition';

 export default {
  components: { VuePartition }
 }
</script>

Or you can use global bean registration:

import VuePartition from '@eneropl/vue-partition';

Vue.use(VuePartition);
