# Demo component

A demo component with sections might look like this:

View code 
<template>
  <VuePartition
    v-model="options"
    :total="total"
  />
</template>

<script>
import { VuePartition } from '@eneropl/vue-partition';

export default {
  components: { VuePartition },
  data() {
    return {
      options: [
        {
          value: 100,
          styles: {
            background: '#FED6BC'
          }
        },
        {
          value: 100,
          styles: {
            background: '#199900'
          }
        },
        {
          value: 100,
          styles: {
            background: '#efca66'
          }
        },
        {
          value: 100,
          styles: {
            background: '#C6D8FF'
          }
        }
      ],
      total: 400
    }
  },
}
</script>

# With minValue & maxValue

A component that represents sections with specified minimum and maximum values ​​that are guaranteed to be kept behind the section.

Conditions

Some conditions you set may be validated by the component and ignored when changed.

View code 
<template>
  <VuePartition
    class="serve__component"
    v-model="options"
    :total="total"
  />
</template>

<script>
import { VuePartition } from '@eneropl/vue-partition';

export default {
  components: { VuePartition },
  data() {
    return {
      options: [
        {
          value: 100,
          maxValue: 150,
          styles: {
            background: '#FED6BC'
          }
        },
        {
          value: 100,
          minValue: 50,
          editable: true,
          styles: {
            background: '#199900'
          }
        },
        {
          value: 100,
          styles: {
            background: '#efca66'
          }
        },
        {
          value: 100,
          minValue: 50,
          styles: {
            background: '#C6D8FF'
          }
        }
      ],
      total: 400
    }
  },
}
</script>
